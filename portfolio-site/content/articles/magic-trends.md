A 5-page concept website for a feature Canva could plausibly ship next: an
AI tool that turns currently-circulating aesthetic trends -- mob wife
winter, clean girl, coastal grandma -- into ready-to-edit color palettes,
font pairings, and layout packs inside Canva's editor.

## Pages

- **Home** -- the pitch, with a live-feeling moodboard teaser
- **How it works** -- the real 4-step pipeline, including an honest note on why it never scrapes TikTok/Instagram/Pinterest
- **Trend gallery** -- 6 real, currently-circulating aesthetic trends, each decomposed into an actual color palette
- **Creators & brands** -- split messaging with different real use cases for each audience
- **Early access** -- a waitlist form and the "why this exists" note

## Why this project exists

Canva's current push is "Magic Studio" -- a family of AI tools (Magic
Design, Dream Lab, Magic Write, Magic Charts) that each take something that
used to require manual design work and make it one click. Magic Trends
applies that same pattern to a real, unsolved gap: aesthetic trends move in
weeks, template libraries move in quarters. This is also a deliberate shift
in format from the rest of this portfolio -- a real static website built to
match the visual craft of an actual product launch page, not a Streamlit
data app.

## Design

Built around the actual subject instead of a generic SaaS template: the
product curates aesthetic trends into moodboards, so the site itself *is*
a moodboard -- rotated pinned cards, real color swatches with hex values,
an asymmetric collage layout. Near-black eggplant canvas, dusty parchment
cards, a hot magenta accent, and an expressive editorial serif (Fraunces)
for display type.

## Stack

HTML, CSS, JavaScript -- no build step, no framework.
